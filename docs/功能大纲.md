## 2. 分阶段交付

- **P1（✅）**：图片生成 + 画风库 + 本地优先存储 + BYOK（前端 IndexedDB + 后端代理）。
- **P2（进行中）**：视频生成（文生视频 / 图生视频 / 首尾帧）；前端页面 + mock + 后端视频代理。
- **P3**：无限画布（tldraw）+ AI 工具箱（扩图/重绘/超分）。
- **P4**：账户体系 + VIP 托管算力（服务端表 users/subscriptions/generations，托管推理服务）。

## 3. 各模块功能清单

### M1 图片生成（✅）
- 文生图、图生图；分辨率/步数/采样器/负面/批量；画风一键应用；结果存本地作品。
- 后端 `POST /api/generate`（BYOK 代理，openai-compatible）。

### M2 视频生成（⏳ P2）
- 文生视频、图生视频、首尾帧；时长/帧率/分辨率/运动强度。
- 后端 `POST /api/video`（BYOK 代理，对应视频端点）。
- 前端 `/video` 页面（mock 占位 → 真实代理）。

### M3 无限画布（⏳ P3）
- 节点/画布编排多步创作；引用图片/视频/参考；项目本地持久化。
- 技术：tldraw（specs 决策）。

### M4 画风库（✅ + 待扩展）
- 内置 23 风格（自有沉淀）、分类 Tab、自定义风格 CRUD、一键应用。
- 待扩展：自定义风格参数细化、预览图（自有资源）、运营改样式。

### M5 本地优先存储（✅）
- IndexedDB：generations / styles_custom / settings / projects；刷新不丢、隐私友好。

### M6 统一模型注册 BYOK（✅）
- 前端：Provider 配置增删改，密钥仅本地、掩码。
- 后端：`POST /api/generate`、`POST /api/video` 代理，密钥仅转发不存储。

### M7 AI 工具箱（⏳ P3）
- 扩图、局部重绘、超分等后处理；复用 M1/M2 的生成能力。

### M8 账户与 VIP 托管算力（⏳ P4）
- 账户注册/登录（JWT）；VIP 不排队、超清出片；托管推理服务。
- 引入服务端库（与本地 IndexedDB 并存）。

### M9 设置与偏好（✅ 基础）
- 模型注册、清空本地数据、主题/路由记忆。

## 4. 非功能需求

- **本地优先 · 零后端**：MVP/P2/P3 数据存浏览器，不上云不登录（产品卖点）。
- **隐私**：API Key 仅存本地/仅转发到用户指定端点，绝不入库/日志（CODEBUDDY.md §3）。
- **性能**：首屏到出图 ≤ 3 步；本地项目刷新不丢。
- **合规**：功能体验对齐，品牌/文案/资产自有化；不复制对标源码/受保护素材（CODEBUDDY.md §2）。
- **可观测**：结构化日志 + traceId；统一返回信封 `{code,message,data,traceId}`。
