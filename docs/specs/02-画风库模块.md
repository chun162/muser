# 02 · 画风库模块

> 实施真源 · P1 模块。依赖：数据契约(06)；被 图片生成(01) 消费。

## 1. 数据结构（Style）

见 `06` 的 `Style` 类型。要点：

| 字段 | 说明 |
|------|------|
| id | 唯一标识 |
| name | 显示名（自有命名，避免照搬受保护名称） |
| category | `2d` / `3d` / `realistic` / `custom` |
| promptTemplate | 风格提示词模板，可含 `{prompt}` 占位 |
| params | 该风格建议的默认参数（resolution/steps/sampler 等） |
| modelHint | 推荐 Provider/模型（可选，供 04 路由参考） |
| preview | 自有预览资源（**不得抓取对标版权图**，见 CODEBUDDY.md §2） |
| builtin | 是否内置（true=内置，false=用户自定义） |

## 2. 分类与入口

- Tab：全部 / 2D动漫 / 3D动漫 / 写实风格 / 自定义风格
- 网格卡片展示；点击"应用"→ 把 `promptTemplate`+`params` 注入图片生成(01) 的输入态。
- "自定义风格" Tab 仅显示 `builtin=false` 的用户风格。

## 3. 内置风格清单（种子，作为自有沉淀起点）

> 以下为**分类框架与风格方向**的种子；具体 prompt 模板与预览需由内容方以**自有资产**填充（不复制对标文案/图片）。

| 分类 | 方向（示例，待重写为自有描述） |
|------|------|
| 2D | 日式动漫、经典手绘动画、国风水墨、韩系条漫、都市言情、国风彩漫、惊悚漫画、美式漫画、复古年代、儿童绘本、二次元仙侠 |
| 3D | 皮克斯风 CGI、赛博未来、国风3D、史诗奇幻、现代3D动漫、游戏仙侠 |
| 写实 | 现代写实、国风写实、生存写实、乡村写实、大唐盛世、未来写实 |

目标：内置 ≥ 20 风格，覆盖上述三类。

## 4. 自定义风格 CRUD（本地）

- 新增：name + category(custom) + promptTemplate + params + 可选预览（用户本地图）。
- 编辑 / 删除：仅限 `builtin=false`。
- 持久化：存 IndexedDB（见 03 的 `styles_custom` store），刷新不丢。
- 一键应用同内置风格。

## 5. 验收标准

- [ ] 四类 Tab 可切换，内置风格全部可见（≥20）。
- [ ] 点"应用"能把模板+参数正确带入图片生成输入态。
- [ ] 自定义风格可增/删/改，刷新后仍在。
- [ ] 内置风格不可被用户误删/误改（只读）。
- [ ] 无任何对标受保护预览图；预览为占位或自有资源。

## 6. 注意

- 画风库的**描述文本与 prompt 模板是轻量自有资产**，应自研沉淀，避免直接搬运对标内容（CODEBUDDY.md §2、docs/03 §E）。
