# 06 · 数据模型与接口契约

> 实施真源 · 全局共享。P1 所有模块的类型/schema/请求契约以此为准。

## 1. 共享类型（TypeScript 示意，待实现）

```ts
// 画风分类
type StyleCategory = '2d' | '3d' | 'realistic' | 'custom';

// 画风（见 02）
interface Style {
  id: string;
  name: string;
  category: StyleCategory;
  promptTemplate: string;      // 可含 {prompt}
  params: GenParams;           // 默认参数
  modelHint?: string;          // 推荐模型(可选)
  preview?: string;            // 自有资源 URL/Blob，禁止对标版权图
  builtin: boolean;            // true=内置(只读)
}

// 生成参数
interface GenParams {
  resolution: string;          // "1024x1024"
  steps: number;               // 默认 30
  sampler?: string;            // 如 dpmpp_2m
  negativePrompt?: string;
  batch: number;               // 默认 1
}

// 单次生成记录（见 03）
interface Generation {
  id: string;
  prompt: string;
  refImage?: Blob;             // 图生图参考(可选,不强制落库)
  params: GenParams;
  styleId?: string;
  resultBlob: Blob;            // 结果图
  createdAt: number;
}

// 项目草稿（P1 占位，P3 启用）
interface Project {
  id: string;
  name: string;
  nodes: unknown[];            // P3 画布节点
  updatedAt: number;
}

// 模型注册（见 04）
type ProviderType = 'openai-compatible' | 'stability' | 'replicate' | 'custom';
interface ModelProvider {
  id: string;
  provider: ProviderType;
  name: string;
  baseUrl: string;
  apiKey: string;              // 仅本地，绝不出库/日志
  defaultModel: string;
  enabled: boolean;
}

// 设置（见 03）
interface Settings {
  theme?: string;
  lastRoute?: string;
  providers: ModelProvider[];  // 密钥引用集合
}
```

## 2. IndexedDB Schema

- 库名：`ciyuan-muse`
- 版本：`DB_VERSION = 1`
- Stores（见 03 §2）：`projects` / `generations` / `styles_custom` / `settings`

## 3. 模型请求/响应契约（图片生成）

**请求（由 04 构造）**
```
POST {baseUrl}/...            // 路径随 provider
Headers: Authorization: Bearer {apiKey}   // 仅本地持有
Body: {
  prompt: string,             // = 用户prompt + 画风模板拼接
  ref_image?: Blob|base64,    // 图生图
  width, height, steps, sampler, negative_prompt, n(batch)
}
```

**响应（归一化后返回 01）**
```
{ images: Blob[] | string[] }   // 多张
```

> 不同 Provider 字段名不同，04 负责**映射到上述统一契约**，模块 01 只认统一结构。

## 4. 画风模板拼接规则

- 选画风时：`finalPrompt = style.promptTemplate.replace('{prompt}', userPrompt)`
- 未选画风：`finalPrompt = userPrompt`
- 参数：画风 `params` 作为默认，用户可在 01 覆盖。

## 5. 错误码（统一）

| code | 含义 | 处理 |
|------|------|------|
| AUTH_INVALID | Key 无效 | 引导设置 |
| RATE_LIMIT | 限流 | 重试/降批量 |
| TIMEOUT | 超时 | 取消+重试 |
| CORS | 跨域 | 提示/可选代理 |
| UNKNOWN | 其他 | 展示原始信息 |
