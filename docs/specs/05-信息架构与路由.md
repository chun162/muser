| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 价值主张 + 四入口卡片（图片可用；视频/画布置灰"即将上线"） |

## 2. 布局组件树（粗略）

```
<App>
  <TopBar> 品牌 + [设置]入口 + 本地优先徽标
  <Routes>
    /            <Home/>        （四入口卡片）
    /image       <ImageGen/>    （输入 + 画布预览 + 结果）
    /styles      <StyleLibrary/>（分类 Tab + 卡片网格）
    /settings    <Settings/> → <ModelRegistry/>

</App>
```

## 3. 全局元素

- **本地优先徽标**：常驻提示"数据存于本机，不上云"。呼应对标卖点。
- **设置入口**：顶部可达模型注册与清空本地数据。
- **空状态**：未配置 Key 时，图片生成引导先去设置配置（见 04）。
